## Getting started

### Running the project

Once you have cloned the project, install dependencies and run the project in development mode:

```bash
cd wjtje.ga
npm install # or yarn
npm run dev
```

This will start the development server on [localhost:3000](http://localhost:3000). Open it and click around.
